# My Calculator

Aplicativo de calculadora desenvolvido com React Native utilizando Expo e React Native Paper.

## Tecnologias Utilizadas

- React Native
- Expo
- React Native Paper
- JavaScript

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Limpar display
- Apagar último caractere
- Layout responsivo
- Tema dark moderno
